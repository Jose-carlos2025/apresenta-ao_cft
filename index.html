Agenda de Contactos em C++

José Carlos Gomes | 1º PI - CFT

Escola Profissional do Fundão - 2024/2025

Introdução

Neste trabalho abordareia acerca de dois topicos: Diagrama de sequências e a implementação de uma Agenda de Contactos usando a linguagem C++, com integração ao banco de dados SQLite para armazenamento eficiente dos dados.

Objetivos

  • Criar um diagrama de sequencias;
  • Desenvolver uma aplicação funcional em C++;
  • Realizar operações CRUD (Criar, Ler, Atualizar e Apagar) em banco SQLite;
  • Construir uma interface de terminal intuitiva;
  • Demonstrar conhecimento na integração entre C++ e SQLite.

Estrutura do Projeto

agenda_contactos/
│
├── build/               → Ficheiros compilados (.exe, .o, etc.)
├── db/                  → Script SQL e base de dados
│   ├── main.sql         → Cria tabela "contactos"
│   └── agenda.db        → Base de dados gerada
├── include/             → Cabeçalhos personalizados (opcional)
├── src/                 → Código-fonte principal
│   ├── main.cpp         → Código principal da aplicação
├── Makefile             → Script de build automático

Dificuldades

  • Configuração do ambiente com SQLite em C++;
  • Tratamento de erros e exceções na manipulação do banco de dados;
  • Estruturação do código para manter organização e modularidade;
  • Implementação de buscas eficientes dentro da base de dados.

Conclusão

O projeto permitiu aplicar os conhecimentos de programação em C++ com uso prático de banco de dados. Superando desafios técnicos e estruturais, consegui entregar uma aplicação funcional e organizada. A experiência contribuiu diretamente para o crescimento como programador.
